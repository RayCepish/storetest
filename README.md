# storetest
Storetest is a demo **application for browsing products/saving favorites and selecting products to cart. Also contains a webview with a Mini Browser page (Google)**


| Layer                     | Technology                                        |
| ------------------------- | ------------------------------------------------- |
| **UI Framework**          | Flutter 3.35.5 (stable channel)                   |
| **Language**              | Dart 3.9.2                                        |
| **State Management**      | BLoC (flutter_bloc)                               |
| **Dependency Injection**  | GetIt + Injectable                                |
| **Local Storages**        | Flutter Shared Preferences                        |
| **Environment Variables** | Flutter Dotenv                                    |
| **Utilities**             | Equatable                                         |
| **UI Helpers**            | Flutter ScreenUtil                                |
| **Mini Browser**          | Flutter WebView                                   |
| **Code Generation**       | Build Runner + Freezed + Json Serializable        |
